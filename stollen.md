# Weihnachtsstollen

## Ansatz  
| 1/2 Päckchen    | frische Hefe                |
| 150 ml          | lauwarme Milch              |
| 1 EL            | Zucker                      |

## Teig  
| 350 gr          | Mehl                        |
| 75 gr           | Zucker                      |
| 1-2 Tüttchen    | Vanillezucker               |
| 2 Messerspitzen | Muskatnuss                  |
| 2 Messerspitzen | Kardamon                    |
| 2 Messerspitzen | Zimt                        |
| 1               | Ei                          |
| 150 gr          | sehr weiche Butter          |

## Füllung  
| 250 gr          | Rosinen                     |
| 6-8 EL          | Rum                         |
| 100 gr          | Zitronat/orangat            |
| 100 gr          | geschälte, gehackte Mandeln |

## Einlage  
| 200 gr          | Marzipan                    |

## Überzug  
| 100 gr          | warme Butter                |
|                 | Zucker                      |
|                 | Puderzucker                 |

- Die Füllung ein Tag zuvor mischen

- Vorteig ansetzen, warmhalten
- 30 Min gehen lassen
- den Teig machen
- 60 Min gehen lassen
- die Füllung zum Teig geben und nochmals kneten
- den Marzipan in den Teig legen
- 60 Min gehen lassen
- 40 Min in den vorgeheizten Ofen bei 170-180° backen

- 10 Min abkühlen lassen
- großzügig mit weicher Butter einpinseln
- mit Zucker bestreuen

- ganz abkühlen lassen
- mit Puderzucker bestreuen
