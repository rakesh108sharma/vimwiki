# Thai Sweet Chilli Sauce

## Rezept

|        | ORIGINAL              |       |        | ABGEÄNDERT      |   
| ------ | --------------------- | ----- | ------ | --------------- |   
| 5      | mittelscharfe Chillis | ----- | 3-4    | Paprikaschoten  |   
|        |                       | ----- | 4-5    | Cayenne Chillis |   
|        |                       | ----- | 1-2    | Tomaten         |   
| 2      | Knoblauchzehen        | ----- | 3-5    | Knoblauchzehen  |   
| 600 ml | Wasser                | ----- | 500 ml | Wasser          |   
| 200 ml | Apfelessig            | ----- | 200 ml | Apfelessig      |   
| 400 gr | Zucker oder Honig     | ----- | 300 gr | Zucker          |   
| 1 EL   | Speisestärke          | ----- | 1 EL   | Speisestärke    |   
|        | Salz                  | ----- |        | Salz            |   

- Knoblauch und Chillis/Tomaten/Paprikaschoten grob schneiden und mixen
- alles aufkochen
- auf gewünschte Dicke einkochen
