# Tomatensuppe

| 1 Kg   | Tomaten               |
| 1-2    | Zwiebeln              |
| 2-3    | Knoblauchzehen        |
| 250 gr | Kartoffel             |
| 700 ml | Gemüsebrühe           |
| 1-2    | Chillis               |
| Prise  | Salz, Pfeffer, Zucker |
|        | Thymian, Oregano      |
|        | (Rosmarin)            |

- Zwiebel und Knoblauch in Olivenöl anschmorren
- Tomaten kurz mit schmorren
- Alles hinzugeben
- 20 min köcheln
- mixen
- mit Petersilie garnieren
