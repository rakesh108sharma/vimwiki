# Rotkohlsuppe  
| 400 gr | Rotkohl                  |
| 100 fr | Kartoffeln               |
| 2      | Zwiebel                  |
| 1 l    | Brühe                    |
|        | Salz, Pfeffer, Muskatnuß |
|        | Meerrettich              |
| 150 ml | Sahner                   |

- Zwiebel dünsten
- Kartoffel + Rotkohl dazu
- dünsten
- Brühe dazu
- Gewürze dazu
- 20 Minuten köcheln
- mixen
- Sahne dazu
