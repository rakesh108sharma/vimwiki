# Wasabi Creme

| 150 gr | gekochte, mehlige Kartoffel |
| 150 gr | Salatgurke                  |
| 1-3 cm | Wasabi                      |
| 1 Tl   | Kräuter nach Wahl           |
| Priese | Salz, Pfeffer               |

- Gurke im Mixer zerkleinern
- alle Gewürze dazugeben
- Kartoffel zerdrücken
