# Ungarische Gulaschsuppe

| 750 gr     | Rindergulasch        |
| 90 gr      | Schweineschmalz      |
| 375 gr     | Zwiebel              |
| 3-8 Zehen  | Knoblauch            |
| 2 L        | Fleischbrühe         |
| 2-3 EL     | Mehl                 |
| 375 gr     | Kartoffeln           |
| 3          | grüne Paprikaschoten |
| 2 kl Dosen | geschälte Tomaten    |
| 2-3 TL     | Paprika rosenscharf  |
| 2-3 TL     | Paprika edelsüß      |
| 4 TL       | Majoran              |
| 3 TL       | Kümmel               |
|            | Salz, Pfeffer        |
| 1 Flasche  | Rotwein              |
| 2-3 TL     | Zucker               |

- Fleisch in Schweineschmalz anschmorren
- Zwiebeln + Kümmel anschmorren (Fleisch vielleicht vorher rausholen)
- Knoblauch kurz dazu
- Mehl anschwitzen + mit Fleischbrühe angießen
- Tomatendose dazu
- Alle Gewürze dazu
- 40 Minuten kochen
- kleingewürfelte Kartoffeln + 1 Glas Rotwein dazu
- 20 Minuten kochen
- Paprika + 1 Glas Rotwein dazu
- 20 Minuten kochen
- abschmecken + 1 Glas Rotwein
