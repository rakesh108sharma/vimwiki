# anaconda

creating/manipulating/removing virtual envirements for python

>
- create env:  
  `conda create --name NAME PACKAGE`  
>
- list env:  
  `conda env list`
>
- change env:  
  `conda activate ENV-NAME`
>
- change back into base-env:  
  `conda deactivate`
>
- remove env:  
  `conda remove --name ENV-NAME --all`
