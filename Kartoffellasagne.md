# Kartoffel Lasagne (türkisch)

| 700 gr | Kartoffel            |
| 1/2 Tl | Salz                 |
| 25 ml  | Olivenöl             |
|        |                      |
| 200 gr | Gehacktes            |
| 1 kl   | Zwiebel              |
| 2-3    | Knoblauch            |
|        | Salz, Pfeffer, Chili |
| 1 El   | frische Minze        |
|        | Petersilie           |
|        |                      |
| 100 gr | Cheddar              |
| 4      | Kirschtomaten        |
|        | Petersilie           |

- Kartoffeln:
    - in dünne Scheiben schaben/schneiden
    - mit Salz und Öl mischen
- Gehacktes mit den Zutaten vermengen
- Lasagne:
    - 1/2 Kartoffeln in eine Schüssel auslegen
    - 1/2 Gehacktes drauf
    - Rest Kartoffeln
    - Rest Gehacktes
    - 30 Minuten auf 230°
- Kruste:
    - Käse und Tomaten auf die Lasagne geben
    - 15 Minuten auf 200 °
